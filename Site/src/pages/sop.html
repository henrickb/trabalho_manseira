Sistema Operacional- Windows_Linux


O que é Sistema Operacional SO?


O SO é uma junção de softwares, com a finalidade de organizar e liderar os recursos de um sistema, desde componentes de hardware e sistemas de arquivos a programas de terceiros, estabelecendo a interface entre o computador e o usuário. Por ele também que é possível conectar outras ferramentas na máquina, ex: mouse, teclado, impressora.


O que ele faz?


O sistema operacional introduz uma camada de abstração entre o hardware e o usuário, que transforma comandos no mouse, teclado e solicitações do sistema, como gerenciamento de recursos (CPU, memória RAM), em linguagem de máquina, enviando instruções ao processador. Este último os traduz para código binário, executa os comandos e envia as respostas como informações que aparecem na sua tela.


Dois exemplos de Sistemas Operacionais são...

Windows



Curiosidade: O Android passou o Windows quando os smartphones se popularizaram. Até o surgimento do Android o Windows ganhava como sistema mais usado no mundo todo independente da plataforma. Entretanto isso mudou com a chegada dos smartphones, hoje o Android é o S.O mais usado, representando 40% do uso global.

O Windows além de ser um dos SO mais conhecidos no mundo, é também considerado o mais fácil de manipular e navegar pela facilidade que se encontra em fazer muitos comandos por clicks em vez de códigos no terminal central do computador. Mas mesmo com as diversas facilidades que existem nele, ainda se ver muitas pessoas preferindo os códigos por achar mais rápido.
